# jamtech-api

## Installation Instructions

- `pip3 install virtualenv`
- `virtualenv env`
- `source env/bin/activate`
- `pip3 install -r reqs.txt`
- `python3 manage.py makemigrations`
- `python3 manage.py migrate`

## How to run server
- `python3 manage.py runserver`
